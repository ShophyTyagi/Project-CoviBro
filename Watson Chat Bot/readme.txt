csv files used to train the chat bot :

CoviBro_intents.csv - This file contains all the intents that are used in Watson Assistant.
CoviBro_entities.csv - This file comtains all the entities that were added in Watson Assistant.
Dialogue Dataset_training.csv - This file contains dataset built from icliniq.com, healthcaremagic.com, healthtap.com and all copyrights of the data belong to these websites. [Source : https://github.com/UCSD-AI4H/COVID-Dialogue.git]

Skills added to the chat bot : 
CoviBro_Skills.json - This is a json file that has the entire dialogue flow of Watson Assistant. This is the skill file that we add to Watson Assistant when we make it.
